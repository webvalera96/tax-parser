# Парсер расчетных такс ПАО "МТС" в формате .docx

Парсер, который умеет раскрывать диапазоны в таблицах документа формата Microsoft Word (.docx) в расчетных таксах ПАО "МТС"

Обрабатывает колонки "Код региона" и "Коды региона" следующим образом:

  - "7432: 1,2,3" &rightarrow; "74321, 74322, 74323"
  - "74321, 74322-74325 &rightarrow; "74321, 74322, 74323, 74324, 74325"

Результат преобразования сохраняется в виде .xlsx файла
